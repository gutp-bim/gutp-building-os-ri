# Issue tracker: GitHub

Issues and PRDs for this repository live in GitHub Issues at
`gutp-bim/gutp-building-os-ri`.

## Conventions

- Create, read, comment on, label, and close issues with the GitHub connector or
  the `gh` CLI.
- Read the full issue body, comments, and labels before changing an issue.
- Publish implementation work as independently verifiable vertical slices.
- Reference parent and blocking issues in each child issue.
- Do not close an umbrella or parent issue until every acceptance criterion is
  demonstrably complete.

When a skill says to publish work to the issue tracker, create a GitHub issue in
`gutp-bim/gutp-building-os-ri`.
