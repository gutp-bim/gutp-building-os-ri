# Triage labels

| Canonical role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is required |
| `needs-info` | `needs-info` | Waiting for information from the reporter |
| `ready-for-agent` | `ready-for-agent` | Fully specified and safe for an AFK agent |
| `ready-for-human` | `ready-for-human` | Human implementation or judgement is required |
| `wontfix` | `wontfix` | The work will not be actioned |

Use the label in the middle column whenever an engineering skill refers to the
corresponding canonical role.
